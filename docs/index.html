概要と免責

免責
  • 個人用のツールなので環境依存の問題には対応できません。
  • リクエストについては基本は応えられないと思います。
概要
  • FGOに関するツールを気分と空き時間に応じて作成しています。
  • 以下の環境で確認しています。よほど古かったり特異でなければ他環境でも利用できると思われます。
    • Google Chrome / Windows 10
    • Google Chrome / Android
  • データの保存先はブラウザになるためブラウザ間や端末間のデータ共有は出来ません。
  • 画面を一度表示した後は通信は発生しません。
その他
  • GitHubでソースを公開しています。機能に問題があればforkしてください。
  • ikisa/fgo_tools
